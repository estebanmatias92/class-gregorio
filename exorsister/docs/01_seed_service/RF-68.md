# RF-68: Desacoplamiento por Interfaz

| Campo | Valor |
|-------|-------|
| **ID** | RF-68 |
| **Módulo** | Servicio de Semilla (SeedService) |
| **Prioridad** | Media |
| **Origen** | CU-01 |
| **Dependencias** | RF-66 |

## Descripción

El sistema debe permitir que cada módulo (Inventario, Combate, Exploración, Misiones, Jefe Final) consuma datos de la semilla sin conocer el algoritmo de generación interno.

## Criterios de aceptación

- SeedService define una interfaz/contrato abstracto que declara todos los métodos de consulta.
- Los módulos clientes dependen solo de la interfaz, no de la implementación concreta.
- Existe una implementación concreta de la interfaz que contiene el algoritmo de generación.
- Existe un mock/fake de la interfaz para que los demás módulos puedan testearse sin instanciar una semilla real.
- Cambiar el algoritmo de generación interno no afecta a ningún módulo consumidor.

## Backlog asociado

### S-11: Definir interfaz/contrato abstracto de SeedService

- **Prioridad:** Media
- **Dependencias:** S-01
- **Descripción:** Definir una interfaz abstracta (clase o trait) que declare todos los métodos del servicio. Los módulos clientes dependen solo de esta interfaz, no de la implementación concreta.
- **Criterios de aceptación:**
  - La interfaz declara: `generateSeed()`, `getEnemyAt()`, `getItemStats()`, `getDailyMission()`, `getEventAt()`, `getMalDelMundo()`, `getRitualPages()`, `getRewardPool()`.
  - Existe al menos una implementación concreta de la interfaz.

### S-12: Proveer mock/fake de SeedService para tests unitarios

- **Prioridad:** Media
- **Dependencias:** S-11
- **Descripción:** Implementar un mock/fake de SeedService que los demás módulos puedan usar en sus tests unitarios. No requiere semilla real; devuelve datos prefijados.
- **Criterios de aceptación:**
  - El mock implementa la misma interfaz que SeedService.
  - Cada método del mock permite configurar el valor de retorno.
  - Los módulos Combate, Exploración, Misiones, Jefe Final pueden testearse con el mock sin instanciar una semilla real.
